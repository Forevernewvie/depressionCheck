# 설문 도구 IP/저작권 상태 (릴리즈 게이트)

## 목적
- 선별(screening) 앱 배포 전 설문 도구의 권리 상태를 명시적으로 관리한다.
- 권리 증빙이 없는 모듈은 사용자 공개를 차단한다.

## 현재 상태

| 도구 | 앱 노출 상태 | 권리 상태 | 배포 조건 |
|---|---|---|---|
| PHQ-2 | 활성 | `UNKNOWN` | 권리자 이용조건/번역 승인 증빙 확보 후 상용 배포 |
| PHQ-9 | 활성 | `UNKNOWN` | 권리자 이용조건/번역 승인 증빙 확보 후 상용 배포 |
| HADS-D | 비활성 | `BLOCKED` | 서면 사용권/허가 확보 전 비활성 유지 |
| CES-D | 비활성 | `BLOCKED` | 권리자/라이선스 확인 전 비활성 유지 |
| BDI-II | 비활성 | `BLOCKED` | 서면 사용권/허가 확보 전 비활성 유지 |

## 필수 증빙 아티팩트
- `docs/legal/contracts/` 하위에 보관:
  - 권리자 허가서(PDF)
  - 상업적 이용 가능 범위 문서
  - 번역본 사용 승인 문서(해당 시)

## 릴리즈 규칙
1. `BLOCKED` 도구는 라우트 접근을 포함해 사용자 화면에서 비활성 상태를 유지한다.
2. `UNKNOWN` 도구는 권리 검증 전까지 법적 고지와 내부 승인 기록이 필요하다.
3. 앱은 진단(diagnosis) 표현을 사용하지 않고 선별(screening)로만 표기한다.
